# Drink-Beer

> 맥주의 평점을 토이 프로젝트

## Stack
- Node.js
  - [Express](http://expressjs.com)
- React
- MongoDB

## Package
- cross-env
- dotenv
- react-router-dom

## How to install
`$ git clone https://github.com/liante0904/drink-beer.git`

`$ yarn dev-install`
## How to use
### Integrated Run
```bash
yarn dev
```
### Separate Run
```bash
yarn server
yarn start
```
