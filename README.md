# Drink-Beer

> 맥주의 평점을 토이 프로젝트

## Stack
- Node.js
  - [Koa](https://koajs.com/)
- React
- MongoDB

## Package
- corss-env
- react-router-dom
- query-string

## How to install
`$ git clone https://github.com/liante0904/drink-beer.git`

`$ yarn dev-install`
## How to use
### Integrated Run
```bash
yarn dev
```
### Separate Run
```bash
yarn server
yarn start
```
