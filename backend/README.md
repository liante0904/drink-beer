# drink-beer-backend
NodeJS, Express, Mongoose(MongoDB)를 이용한 REST-API 예제 

The `api` uri preceeds all API endpoints and the following endpoints are currently available
* GET `/api/beers`
* GET `/api/beers/:id`
* POST `/api/beers`
* PUT `/api/beers/:id`
* DELETE `/api/beers/:id`
